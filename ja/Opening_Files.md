## ライブ プレビュー ワークフロー

1. Markdown ファイルを Marked にドラッグします (または {% appmenu File, Open... ({{cmd}}O) %} を使用します)。
2. 好みのエディタでファイルを編集します。
3. 保存 --- マークするとプレビューが自動的に更新されます。

Vault の監視、ストリーミング プレビュー、エディター固有のガイドについては、[Mac でのライブ Markdown プレビュー](Live_Markdown_Preview_on_Mac.html) を参照してください。

## ドックアイコンにドラッグ

すでに編集中のファイルでマークを使用する最も簡単な方法は、エディタのツールバーまたは Finder から Dock のマーク アイコンにドキュメント アイコンをドラッグすることです。 Marked は、そこにドロップされた Markdown ファイル (またはテキスト ファイル) の追跡をすぐに開始します。 Finder からファイルを直接ドラッグすることもできます。

## メニューの使用

![][2]

   [2]:images/file_open.jpg @2x width=300px height=118px class=center

もちろん、{% appmenu File, Open... ({{cmd}}O) %} メニュー オプションを使用して Markdown ファイルを直接開くこともできます。 Marked はテキスト エディタを使用しなくても正常に動作します。クリックするだけで Markdown をプレビューして変換できます。

Marked では、**`.rtf` および `.rtfd`** ファイルを直接開くこともできます (たとえば、Pages または TextEdit からのエクスポート)。元のアプリで保存すると、プレビューおよび更新のために Markdown に変換されます。画像や制限事項などの詳細については、[RTF および RTFD のサポート](RTF_Support.html) を参照してください。

Marked は、**`.pdf`** ファイルを同じ方法で開くことができます。変換はバックグラウンドで実行され、完了するとプレビューが更新されます。これは、短いテキストベースの PDF に最適です。大きなマニュアルやスキャンした文書は速度が遅くなり、精度も低くなります。詳細と制限事項については、[PDF サポート](PDF_Support.html) を参照してください。

## クリップボードから

互換性のある (Markdown など) テキストがクリップボードにある場合は、{% appmenu File, New, Clipboard Preview ({{shift}}{{cmd}}V) %} を選択してインスタント プレビューを開くことができます。 Web ブラウザーまたは HTML または RTF をクリップボードに置く他のアプリから選択範囲をコピーした場合、Marked はそれをプレビュー用に Markdown に変換します。 TextEdit や Pages などのアプリから RTF を貼り付けると、大きいフォント サイズは見出しレベルに大まかに変換されます (たとえば、非常に大きいテキストはレベル 1 の見出しになり、小さい大きいテキストはレベル 2 になります)。複数のクリップボード プレビューを一度に開くことができ、{% appmenu File, Save Transient Preview %} を選択して新しいファイルに保存できます。

## 新しいドキュメントの作成

マークを付けると、{% appmenu File, New ({{cmd}}N) %} コマンドを使用して新しい空のテキスト ファイルを作成できます。ファイルを保存する場所をすぐに尋ねられます。[テキスト エディター] ボタンの横にある {% prefspane Apps %} で [新しいファイルを自動的に編集する] オプションを有効にすると、新しく作成されたファイルがデフォルトのエディターで自動的に開きます。

## 最近開いたものを開く

![][3]

   [3]:images/open_recent.jpg @2x width=300px height=174px class=center

Marked は最近のドキュメントも追跡します。 {% appmenu File, Open Recent %} メニュー オプションを使用すると、開いていたファイルが表示され、そのファイルにジャンプして戻ることができます。 {% kbd shift cmd R %} を使用すると、最後に表示していたファイルをすぐに再度開くことができます。 {% kbd shift cmd P %} または [クイック アクション] (Quick_Actions.html) を使用して、メニューを実行し、キーボードからコマンドをプレビューします。 他にも多くのキーボード ショートカットがあります。それらを学びたい場合は、[キーボード] の下に表を見つけることができます。ショートカット](Keyboard_Shortcuts.html)。
