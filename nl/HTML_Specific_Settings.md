## Automatische header-ID's uitschakelen

Marked bevat een optie om het automatisch genereren van header-ID's uit te schakelen. Deze optie vind je in de {% prefspane Processor %}.

## Willekeurige voetnoot-ID's

In het paneel **Processor** kunt u het selectievakje "Willekeurige voetnoot-ID's gebruiken" aanvinken om willekeurige voetnoot-ID's te genereren, waardoor conflicten worden voorkomen wanneer meerdere documenten op één webpagina worden weergegeven. Deze optie is alleen beschikbaar bij gebruik van de MultiMarkdown processor.

## Proces Markdown Binnenkant van HTML

Standaard wordt de syntaxis Markdown binnen HTML bloktags doorgaans genegeerd door Markdown processors. Deze optie dwingt Marked om door te gaan met de verwerking binnen blokelementen. Houd er rekening mee dat sommige markeringen problemen kunnen veroorzaken.
