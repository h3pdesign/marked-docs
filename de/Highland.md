# <%= @title %>

[Highland 2][highland] Benutzer können eine Vorschau der Fountain- oder Markdown-Entwürfe in Marked anzeigen, indem sie **das Highland-Dokument** aus dem Finder (oder vom Proxy-Symbol in der Titelleiste des Fensters) auf das Symbol von Marked oder in ein geöffnetes Marked-Fenster ziehen.

Marked aktualisiert die Vorschau, wenn sich die Datei auf der Festplatte ändert, genau wie eine normale Markdown-Datei.

[highland]: https://quoteunquoteapps.com/highland-2/
