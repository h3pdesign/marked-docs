# <%= @title %>

Ziehen Sie ein [MindNode][mn] Dokument auf Marked (oder verwenden Sie den MindNode-Befehl **In Marked** / Vorschau öffnen, je nach MindNode-Version). Marked behandelt die Datei wie den Markdown-Export von MindNode, sodass die Hierarchie Ihrer Map in der Vorschau zu Überschriften und Listen wird.

## Streaming-Vorschau (MindNode 2026.3+)

I> **MindNode 2026.3** und neuer fügen Marked [streaming preview](Streaming_Preview.html) hinzu. Wählen Sie in MindNode **Ansicht → Vorschau in Marked**, um es zu öffnen; Die Vorschau wird live aktualisiert, wenn Sie die Karte ändern.

## MindNode Classic im Vergleich zu MindNode 7+

W> Die aktuelle Integration zielt auf **MindNode Classic**-Bundles ab. Neuere MindNode-Versionen verwenden ein anderes Paketformat; Wenn das Öffnen einer Map fehlschlägt oder Marked ein **„Bitte von MindNode öffnen“**-Blatt anzeigt, bereiten Sie die Map über das MindNode-Menü **Erweitert** vor (häufig **In einer anderen Anwendung öffnen** / Marked) oder exportieren Sie ein kompatibles Bundle, anstatt ein rohes `.mindnode`-Paket zu ziehen, das nicht für die Vorschau vorbereitet wurde.

Da die Datei von MindNode vorbereitet wird, bevor Marked sie liest, speichern Sie sie immer in MindNode, bevor Sie mit einer Aktualisierung in Marked rechnen.
