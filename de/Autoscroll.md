# <%= @title %>

Durch automatisches Scrollen wird die Vorschau langsam verschoben, sodass Sie freihändig lesen oder Korrekturen vornehmen können. Es ist getrennt von [Table of Contents](Document_Navigation.html) und [Mini Map](Document_Navigation.html#minimap); Es scrollt automatisch durch die aktuelle Ansicht.

## Starten und Stoppen

Lösen Sie den automatischen Bildlauf aus, indem Sie {% kbd s %} drücken. Die Vorschau scrollt mit der aktuellen Geschwindigkeit vorwärts.

Unten links zeigt eine Anzeige die Geschwindigkeit an. Passen Sie es mit den Pfeiltasten **Auf** und **Ab** an; Halten Sie **Shift** mit den Pfeilen gedrückt, um größere Schritte zu machen.

Return ({% kbd return %}) pausiert den Bildlauf oder setzt ihn fort.

{% kbd shift s %} kehrt die Bildlaufrichtung um, während der automatische Bildlauf aktiv ist.

Drücken Sie erneut {% kbd s %} oder Escape, um den Autoscroll-Modus zu verlassen.

Wenn Sie Marked wie einen Teleprompter verwenden, können Sie skriptgesteuerte Pausen in der Markdown-Quelle hinzufügen – siehe [Autoscroll pauses](Special_Syntax.html#pauses) unter [Marked Special Syntax](Special_Syntax.html).
