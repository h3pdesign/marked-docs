# <%= @title %>

Autoscroll bewegt die Vorschau langsam weiter, sodass Sie freihändig lesen oder Korrektur lesen können. Die Funktion ist unabhängig vom [Inhaltsverzeichnis](Document_Navigation.html) und der [Mini-Übersicht](Document_Navigation.html#minimap); sie scrollt die aktuelle Ansicht automatisch.

## Starten und Stoppen

Starten Sie Autoscroll, indem Sie {% kbd s %} drücken. Die Vorschau scrollt mit der aktuellen Geschwindigkeit vorwärts.

Unten links zeigt eine Anzeige die Geschwindigkeit an. Passen Sie sie mit den Pfeiltasten **Auf** und **Ab** an; halten Sie **Shift** mit den Pfeiltasten gedrückt, um größere Schritte zu machen.

Return ({% kbd return %}) pausiert den Bildlauf oder setzt ihn fort.

{% kbd shift s %} kehrt die Bildlaufrichtung um, während Autoscroll aktiv ist.

Drücken Sie erneut {% kbd s %} oder Escape, um den Autoscroll-Modus zu verlassen.

Wenn Sie Marked wie einen Teleprompter verwenden, können Sie skriptgesteuerte Pausen in der Markdown-Quelle hinzufügen – siehe [Autoscroll-Pausen](Special_Syntax.html#pauses) unter [Spezielle Marked-Syntax](Special_Syntax.html).
