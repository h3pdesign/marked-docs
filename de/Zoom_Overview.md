# <%= @title %>

Zoom-Übersicht verkleinert den Vorschautext, sodass Sie mehr vom Dokument auf einmal sehen können – nützlich, um Strukturen zu erkennen, nach einem Abschnitt zu suchen oder die Länge zu beurteilen, ohne das [Mini Map](Document_Navigation.html#minimap) öffnen zu müssen.

## Zoom-Übersicht verwenden

Drücken Sie {% kbd z %} in der Vorschau, um herauszuzoomen. Drücken Sie {% kbd z %} erneut, um zur normalen Größe zurückzukehren. Während des Zoomens können Sie auch **Escape** drücken oder die Schaltfläche zum Verwerfen am {% kbd z %}-Tipp verwenden, um wieder hineinzuzoomen.

## Visualisierung von Wortwiederholungen

Die gleiche {% kbd z %}-Verknüpfung gilt, wenn **Wortwiederholung** ({% kbd ctrl cmd w %}) geöffnet ist: Durch Umschalten des Zooms können Sie sehen, wo sich wiederholte Wörter im Dokument anhäufen.

## Verwandt

Eine skalierte Miniaturansicht der gesamten Seite (mit optionalen Lesezeichenmarkierungen) finden Sie unter [Document Navigation](Document_Navigation.html#minimap).
