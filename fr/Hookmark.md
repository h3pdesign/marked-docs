# <%= @title %>

Les liens [Hookmark](https://hookproductivity.com/) peuvent pointer vers des fichiers et des ressources auxquels votre Markdown fait référence, y compris des images.

## Résolution des URL Hookmark [hookmark-url-resolution]

Marked peut résoudre les liens `hook://` pour les images et autres éléments d'un fichier lorsque l'intégration Hookmark est activée.

## Activer dans les paramètres [enable-in-settings]

Ouvrez {% prefspane Apps %} et activez l'option liée à Hookmark. Une fois activé, Marked résout les URL Hookmark lors du rendu afin que les ressources liées apparaissent normalement dans l'aperçu et l'exportation.
