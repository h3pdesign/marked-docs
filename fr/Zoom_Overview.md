# <%= @title %>

Zoom Aperçu réduit le texte d'aperçu afin que vous puissiez voir une plus grande partie du document à la fois, ce qui est utile pour repérer la structure, numériser une section ou juger de la longueur sans ouvrir la [Mini-carte](Document_Navigation.html#minimap).

## Utilisation de Zoom - Présentation

Appuyez sur {% kbd z %} dans l'aperçu pour effectuer un zoom arrière. Appuyez à nouveau sur {% kbd z %} pour revenir à la taille normale. Pendant le zoom, vous pouvez également appuyer sur **Échap** ou utiliser la commande de rejet de l'astuce {% kbd z %} pour effectuer un zoom avant.

## Visualisation de la répétition de mots

Le même raccourci {% kbd z %} s'applique lorsque **Répétition de mots** ({% kbd ctrl cmd w %}) est ouvert : activer le zoom vous aide à voir où les mots répétés se regroupent dans le document.
